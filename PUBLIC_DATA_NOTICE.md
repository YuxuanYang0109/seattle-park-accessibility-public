# Public data notice

This repository is a privacy-minimized public demonstration of the Seattle Park Accessibility project.

Published content includes official or aggregate geographic data, Census Block Group aggregates, derived accessibility scores, generalized presentation assets, and aggregate park ratings.

The public release excludes individual reviewer names, original review identifiers, review text, owner replies, avatar URLs, remote photo URLs, and locally cached review photographs. The repository was created with a fresh Git history and does not contain the private research repository's earlier file history.

Interactive geographic data delivered to a browser can still be downloaded by a visitor. Do not add confidential, licensed-restricted, or personally identifiable data to the public deployment.
