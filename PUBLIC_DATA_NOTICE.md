# Public data notice

This repository is a privacy-minimized public demonstration of the Seattle Park Accessibility project.

Published content includes official or aggregate geographic data, Census Block Group aggregates, 15-minute route-derived accessibility scores, generalized presentation assets, aggregate park ratings, and explicitly synthetic text used by the comment-rain and text-landscape scenes.

The public release excludes individual reviewer names, original review identifiers, review text, owner replies, avatar URLs, place identifiers, remote photo URLs, locally cached review photographs, scraping manifests, and private source files.

Interactive geographic data delivered to a browser can still be downloaded by a visitor. Do not add confidential, license-restricted, or personally identifiable data to the public deployment.
